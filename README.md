## Virtual Env. Setup
### install
pip install virtualenv [--user]

### create an env
virtualenv myenv
virtualenv -p /usr/local/bin/pypy myenv # using the pypy distribution

### use the env
source myenv/bin/activate

### exit the env
deactive # which is usable only after you activate the env
